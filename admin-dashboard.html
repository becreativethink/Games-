⚙ Admin Settings
System configuration and admin access
Firebase Database Rules
Copy these rules into your Firebase Realtime Database → Rules tab to secure user and admin data.
{
  "rules": {
    "users": {
      "$uid": {
        ".read":  "$uid === auth.uid",
        ".write": "$uid === auth.uid"
      }
    },
    "adminAPI": {
      "$uid": {
        ".read":  "$uid === auth.uid",
        ".write": false
      }
    },
    "adminConfig": {
      ".read":  false,
      ".write": false
    }
  }
}
Register Admin UID
Add a Firebase user UID to adminConfig/adminUids/<uid> = true. Registered admins will see the Admin nav button in the main app.
⚠ Admin Password Notice
The admin password is hashed and compared client-side as a security layer. For production deployments, consider moving admin authentication to a backend server with Firebase Admin SDK for maximum security. The admin dashboard URL should also be kept private.
